# DC Inside Board Telegram Bot
This bot reads messages in a DC Inside board and sends them to you.

# How to run

Create a bot. Using Bot Father, get the bot ID.
Send a message "/start" to the bot.
Then, get a chat ID, also.

A reference is here:
